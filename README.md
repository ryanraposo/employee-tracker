# Employee Tracker

Track employees via a database-connected, prompt-based terminal app.  

## Requirements

- Node.js

## Installation

    $ git clone https://github.com/ryanraposo/employee-tracker
    $ cd employee-tracker
    $ npm install

## Usage

    $ cd employee-tracker
    $ node app
